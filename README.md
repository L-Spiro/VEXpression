# ExpressionEvaluator
A Coding Buddy for Easy Code Prototyping

## Overview
C-like syntax with Python mixed in for convenience. No need to include headers, no need to code a `main()` body, no need to hit a button to compile—just start typing and the result of your code is shown in real-time as you type (REPL).

The language is not meant for creating large projects; its focus is entirely on easy coding and fast results. It can be used for tasks as simple as decimal<->binary<->octal<->hex conversions or as a basic calculator to tasks as complex as advanced mathematical calculations or complex C functions.

## Feature Support
### General
Unsigned integers, signed integers, floating-point values, SIMD registers, vectors, strings, and maps.<br>
Hundreds of built-in functions from several domains (DSP, color space conversions, numerical integration, string processing, etc.)<br>

### C/C++
All of <cmath\>.<br>
Operator precedence.<br>
SIMD (AVX, AVX2, AVX-512, MMX, SSE, SSE2, SSE3, SSE4.1, SSE4.2, NEON, MIPS, etc.)<br>

### Python
Many functions from SciPy and NumPy.<br>
All String, Array, and Dictionary methods.<br>
